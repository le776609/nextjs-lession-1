# Complex Screen Plan

Mục tiêu: thêm một màn hình Next.js mới với nhiều animation, cảm giác dày đặc và phức tạp, nhưng vẫn chỉ dùng CSS + React cơ bản.

## 15 task nhỏ
1. Tạo tài liệu kế hoạch 15 bước triển khai.
2. Khởi tạo route mới `/command-center` với khung trang cơ bản.
3. Tạo dữ liệu mẫu cho các khối dashboard.
4. Thêm background nhiều lớp và hiệu ứng ambient.
5. Thêm header/hero của command center.
6. Thêm cụm metric cards nổi.
7. Thêm radar grid và status feed.
8. Thêm timeline mission.
9. Thêm energy rails và pulse bars.
10. Thêm drone fleet panel.
11. Thêm action dock và navigation cuối trang.
12. Thêm bộ keyframes animation dùng chung.
13. Thêm toàn bộ style layout/visual cho màn command center.
14. Nối route mới vào trang chủ + cập nhật README.
15. Responsive + accessibility polish + build kiểm tra cuối.
